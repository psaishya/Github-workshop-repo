ALGORITHM OF THE PROGRAM:

The algorithm of the program is:

  • Start.
  • Display the introduction of the project, by calling the introduction function.
  • Ask if user wants to continue.
  • Open a CSV file.
  • Check if the CSV file is opened or not by checking if the provided path of the CSV file is correct.
  • Read each character from the file and display the contents of file.
  • Count the number of rows of data from file and store in n_data.
  • Declare an array of structure(previously declared) value_arr of size n_data.
  • Skip the line 1 of the file.
  • Get each character from file until the end of file indication.
  • Store the expected data in expected member of ith element of array and actual data in actual member of ith element of array.
  • Call the count function.
  • Compare the values stored in each member of array with ‘Y’ and ‘N’, and increase the values of TP, FP, FN, accordingly.
  • Display the values of TP, FP, and FN.
  • Display the menu and ask user to make choice, on what they want to calculate.
  • Call the fprecision function if they want to calculate precision.
  • Call the frecall function if they want to calculate recall.
  • Call both the function if they want to calculate both.
  • Call display function if they want to display the contents of file again.
  • Quit the program with the thanking message if they want to quit.
  • Keep displaying the menu and perform calculations until the user chooses to quit the program.
  • Stop.
